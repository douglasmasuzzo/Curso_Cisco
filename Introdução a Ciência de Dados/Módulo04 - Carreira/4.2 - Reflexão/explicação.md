>> ## Módulo04 - Carreira em Análise de Dados( **4.2.1 - Reflexão do Módulo** ) ##

## Descrição ##
 Neste módulo, você compreendeu as funções e responsabilidades de várias carreiras em análise de dados e as respectivas etapas nesta trajetória.

 - _Preparação da carreira de Analista de Dados_
    - Os profissionais de dados desempenham três funções principais nas empresas: **Analista de Dados | Engenheiro de Dados | Cientistas de Dados**. Algumas ferramentas e habilidades comumente mencionadas em anúncios de emprego para   cargos de níveis básicos incluem:
        
        - Atenção em detalhes
        - Habilidades sólidas de comunicação verbal e escrita
        - Capacidade de trabalho cooperativo
        - Proficiência com tabelas ( Microsoft Excel )
        - Conhecimento com Banco de Dados ( SQL )
        - Experiência com linguagens de Programação Orientada a Objetos ( POO )
        - Familiariedade com ferramentas de visualização e exibição

    - _Objetivo_ : Diferenciar entre os demais tipos de funções na Análise de Dados 

 - _Etapas_
    - Um portifólio é um site com exemplos demonstrados do seu trabalho em nível básico. Os portifólios são uma maneira profissional de exibir seus interesses e habilidades. A medida que desenvolve projetos e artefatos para seu protifólio, documente todo o processo de desenvolvimento, incluindo:

        - Perguntas a serem respondidas ou problemas a serem resolvidos
        - Como você escolheu os dados embasados em sua análise
        - Métodos utilizados para analisar os dados
        - Observações e conclusões pessoais 
        - Relatórios e apresentações

>> ## Módulo04 - Carreira em Análise de Dados( **4.2.1 - Quiz : Carreira em Análise de Dados** ) ##

### _Questões Práticas_ ###

 1. _Qual é a função de um Analista de Dados?_
    
    > Resposta : **Consultar e processar dados, fornecer relatórios, resumir e visualizar dados**

 2. _Qual é a função de um Engenheiro de Dados?_
    
    > Resposta : **Criar e operacionalizar pipelines de dados para coletar e organizar dados**

 3. _Qual é a função do Cientistas de Dados?_

    > Resposta : **Aplicar Estatísticas, Machine Learning e Abordagens analíticas para responder a questões críticas de negócios usando técnicas de visualização**
 
 4. _Qual ferramenta é normalmente usada para demonstrar as habilidades de análise de dados a um empregador em potencial?_

    > Resposta : **Portifólio**

 5. _Quais são as três habilidades típicas de um Analista de Dados de nível básico?_

    > Resposta : **Proficiência com Planilhas | Fortes habilidades de Comunicação Verbal/Escrita | Familiaridade com ferramentas de Visualização**
