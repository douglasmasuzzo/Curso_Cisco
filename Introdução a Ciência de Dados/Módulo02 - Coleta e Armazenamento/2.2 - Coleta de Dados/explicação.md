## Descrição ##
 Neste módulo, abordamos o conceito referente sobre Big Data, as características associaidas a Big Data, o benefício do crescimento de dados e as fases de um Pipeline de Dados.

### _Noções Básicas sobre Big Data_ ###

 - Objetivo
    - Descrever o conceito "Big Data" e seus possíveis benefícios.

 O Big Data é um termo usado para **descrever os grandes volumes de dados digitais**, coletados e processados. As características associadas ao termo modificaram a forma de como os dados são obtidos, transmitidos, armazenados e acessados, além da definição dos fundamentos principais que constituem esse modelo : Os "**4 V's**".
 Os fatores que impulsionaram o aumento do crescimento de dados são o aumento de dispositivos IoT, juntamente com o crescimento de dispositivos eletrônicos conectados a Internet e a popularização das redes sociais.  

### _Noções Básicas sobre o Gerenciamento de Big Data_ ###
 
 - Objetivo
    - Descrever o fluxo de dados por meio de pipelines

 Os engenheiros de dados gerenciam os conjuntos de informações por meio de um Pipeline de dados. O termo apresenta três estágios de funcionamento: **Ingestão | Transformação | Armazenamento**. Assim, a definição e a operação desses modos de estágios são essenciais para a realização e desenvolvimento de análises. 
 Os profissionais trabalham utilizam duas fontes principais de captura de dados: **Ingestão em Lote** ( coleta de dados estruturados ) e **Ingestão em Streaming** ( coleta de dados não-estruturados ). A **Transformação** filtra informações corrompidas ou incorretas, convertendo-as para sua versão corrigida e adequada, permitindo contabilizar entre os diferentes tipos de dados. Os dados podem ser alocados digitalmente, remotamente ou em modelos híbridos.

>> ## Módulo02 - Coleta e Armazenamento de Dados ( **2.2.1 - Reflexão** ) ##

## Descrição ##
 Os restaurantes estão utilizando o BigData para personalizar a experiência de seus respectivos clientes. Os dados são coletados por meio de um aplicativo, atendimento drive-thru e sessões de feedback em menus digitais. Esses dados propõem ajustar o nível de experiência que os consumidores alcançam, seguindo as interações e as bonificações atingidas para aprimorar a experiência e fortalecer a fidelidade.
 As sessões de feedback são alteradas conforme o clima, a hora e os eventos locais. O menu visa promover itens e serviços estrategicamente benéficos para impulsionar o desempenho de vendas. Em um dia com o clima abafado, é possível ver a exibição de produtos comercializados neste período. Em outros dias, a empresa divulga a exibição de outros produtos favoráveis ao clima atual. Deste forma, as estratégias aplicadas buscam incitar a atenção e interesses dos consumidores.

### _Questões Práticas_ ###
 
 1. _Os 4 V's do Big Data são descritos como : Volume | Velocidade | Variedade e Veracidade. Sobre o quisito acima, o que se entende por Variedade?_

    > Resposta : **O tipo de dados que não está pronto para processamento e análise**

 2. _Qual termo de Big Data inclui a redução quantitativa de limpeza de dados necessariamente?_
    
    > Resposta : **Veracidade** 

 3. _Qual é a ordem correta dos estágios no PipeLine de Dados?_

    > Resposta : **Ingestão | Transformação | Armazenamento | Análise**
