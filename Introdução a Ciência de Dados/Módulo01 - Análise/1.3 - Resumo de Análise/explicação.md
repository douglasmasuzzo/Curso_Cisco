## Descrição ##
 Neste módulo, você aprendeu sobre as várias formas de dados, como as visualizações de dados são criadas e maneiras de comparar dados e avaliar o valor obtido por meio de análises.

### _O que são Dados?_ ###
 Os dados são criados por meio de muitas atividades diárias modernas. As empresas coletam esses dados e aplicam a análise de dados para fundamentar as aplicações práticas de negócios. 
 Existem três tipos principais de dados : _Dados Observados | Dados Voluntários | Dados Inferidos_. A visualização de dados correta pode apresentar intuitivamente padrões e tendências de dados complexos. Os principais fatores a serem considerados ao escolher uma visualização são: 

 - Objetivo
    - Identificar aplicações de dados na vida diária

 - O número de variáveis a serem mostradas
 - O número de pontos de dados ou unidade de informações em cada variável
 - Se os dados ilustram as mudanças ao longo do tempo
 - A necessidade de fazer uma comparação ou correlação entre diferentes grupos de dados 

### _Os dados estão ao nosso redor_ ###
 A análise de dados começa com a compreensão dos tipos de dados. Os dados são definidos como quantitativos ou qualitativos. 
 Os dados quantitativos e qualitativos são divididos em dados discretos e contínuos. O tipo de dados informa a um sistema como interpretar o valor dos dados, para que o sistema possa realizar operações para transformar e usar os dados em cálculos. 

 Além disso, os dados devem ser categorizados como estruturados ou não-estruturados antes de serem processados, armazenados e analisados. Depois que os dados são definidos e selecionados, eles se tornam relevante para determinar as perguntas à serem respondidas.

 - Tipos de Dados
    - String
    - Integer ( número inteiro )
    - Float ( número decimal )
    - Date 
    - Boolean ( operador lógico )

### _Compreensão dos Negócios_ ###
 A ciência de dados possibilita que as empresas compreendam melhor o impacto de seus produtos e serviços, ajustem seus métodos e objetivos, e forneçam aos seus clientes produtos de melhor qualidade de forma mais rápida. A análise de tendências é uma maneira de obter insights sobre os principais indicadores de desempenho ( KPI ) ao longo do tempo.

 Organizações Humnitárias utilziam dados para servir às suas comunidades e ao mundo. A análise preditiva pode concentrar os esforços humanitários em ações preventivas, em vez de reativas. As agências ambientais rastreiam os dados das mudanças climáticas por meio de observações que permitem previsões de impacto social.
